Class PublisherApi

Service Description: The service that an application uses to manipulate topics, and to send messages to a topic.

<

p>This class provides the ability to make remote calls to the backing service through method calls that map to API methods. Sample code to get started:

<code>
$publisherApi = new PublisherApi();
$topicName = PublisherApi::formatTopicName("(your project here)", "my-topic");
$topic = $publisherApi->createTopic($topicName);
</code>

<
